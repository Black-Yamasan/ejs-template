# ejs-template
テンプレートエンジン「EJS」を使用したテンプレート

## commands

### build for development

```bash
npm run build:dev
```

or

```bash
yarn run build:dev
```

### build for production

```bash
npm run build:prod
```

or

```bash
yarn run build:prod
```

### run server for development

```bash
npm run start
```

or

```bash
yarn run start
```

### run test for javascript

```bash
npm run test
```

or

```bash
yarn run test
```
